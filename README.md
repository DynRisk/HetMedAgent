# HetMedAgent
Slide video presentation for "Why Specialist Models Still Matter: A Heterogeneous Multi-Agent Paradigm for Medical Artificial Intelligence" (ICML 2026)

This video walks through the HetMedAgent framework — a heterogeneous medical multi-agent system that orchestrates generalist LLMs, domain-specific specialist models, and clinicians for clinical decision-making. Topics covered include the system architecture, conflict-aware evidence fusion, uncertainty-based routing, adaptive threshold calibration, and experimental results on cardiovascular clinical tasks.
